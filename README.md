1. Clone the repository:

   ```bash
   git clone https://github.com/ChaikamonS/CountriesOfTheWorld.git

2. Navigate to the project directory

   ```bash
   cd your-project

3. Install dependencies

   ```bash
   npm install

## Running the Application

To start the development server, run

   ```bash
   npm run dev
